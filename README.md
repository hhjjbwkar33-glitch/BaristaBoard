# BaristaBoard

## 概要
BaristaBoard は、カフェのシフト作成を自動化するPythonベースのスケジューリングシステムである。

## 背景・解決したい課題

### 現場の声
> 「誰かが休憩に入ったタイミングであるポジションにつく人が変わり、複雑なパズルのようで考えるのが難しい」

> 「毎日様々な従業員がシフトインするのでそれも考慮しなければならない」

### 具体的な課題
カフェのシフト作成には以下の制約があり、手作業では多大な時間と労力がかかっている。

**① ポジション割り当て**
バリスタ・キャッシャー・フロアの3ポジションを、過不足なく割り振る必要がある。
特に、ピークタイムとなる12:00~14:00においてはバリスタ2人、キャッシャー1人が要求されるという例外がある。

**② ローテーション**
最大で一つのポジションを2時間連続で続けると、ポジションを担当する人が交代する。

**③ 休憩管理**
勤務時間に応じた休憩をシフト表に組み込む必要がある。
- 5時間以上6時間未満：15分休憩
- 6時間以上7時間未満：30分休憩
- 7時間以上：45分休憩

これらの制約が複合的に絡み合うため、特に休憩による人員変動が発生するタイミングでの調整が非常に複雑になる。
